# 🚀 Mini-Landing Page: SoyTuChat → ZonaCreadores

Este repositorio contiene el código fuente de la mini-landing page estática desplegada en el dominio `soytuchat.com`. El propósito principal de este sitio es actuar como un puente de redirección estratégica y canalización de tráfico hacia la plataforma principal `zonacreadores.com`, garantizando un traspaso de usuarios fluido, persuasivo y optimizado.

---

## 🎯 Propósito del Proyecto

El objetivo de este desarrollo es capturar de forma orgánica el tráfico que ingresa buscando soluciones de asistentes virtuales bajo la marca **SoyTuChat** e integrarlo al ecosistema de herramientas extendidas de **ZonaCreadores** (Inteligencia Artificial, automatizaciones, generación de carruseles, etc.).

### Flujo de Experiencia:
1. El usuario llega a `soytuchat.com`.
2. Encuentra el mensaje de cambio de dominio indicando el cambio de Branding a ZonaCreadores.
3. Al interactuar con los botones de llamado a la acción (CTA) o esperar 5 segundos, es redirigido inmediatamente a la plataforma principal.

---

## 🛠️ Arquitectura y Tecnologías

Para garantizar un tiempo de carga ultra-rápido (*First Contentful Paint* óptimo) y reducir costos de infraestructura, el proyecto se construyó sin frameworks pesados:

- **HTML5 Semántico:** Para una estructura limpia y un SEO optimizado.
- **CSS3 Nativo:** Layout completamente *responsive* (Mobile-First) utilizando variables de entorno y flexbox/grid nativos para evitar hojas de estilo masivas.
- **Vanilla JavaScript (ES6+):** Scripts mínimos para el conteo de la redirección.
